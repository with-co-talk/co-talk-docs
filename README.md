# Co-Talk

대화에 집중한 커뮤니케이션 플랫폼

## 개요

Co-Talk은 **대화에 집중한 커뮤니케이션 플랫폼**입니다.

- **Couple Talk**: 커플 간의 깊은 대화와 소통
- **Community Talk**: 공통 관심사를 가진 집단 간의 의미 있는 대화
- **Co-work Talk**: 협업과 업무를 위한 효율적인 커뮤니케이션
